Utilities

The Utility classes are a set of commonly used non-HPI classes.

List
A List is used to store a list of objects.
ListItem
A ListItem stores one object reference within a List.
ListIterator
A ListIterator is used to iterator through a List.
StringBuffer
The StringBuffer maintains a set of character buffers. Code segments use StringBuffer to obtain a temporary character buffer. The StringBuffer cycles through its character buffers which are never deallocated. This is useful when a function wishes to return a char* without requiring the caller to free the memory.
Time
Contains some static methods to obtain the current time in milliseconds and to convert millisecond time to a printable string.
Timer
The Timer class is used for callback notifications based upon time. It maintains an accuracy of one second. The add() methods are used to install a callback function. When a callback function is added, a unique ID is returned. That ID can be used to pause/resume the timer for that callback or to remove the function callback timer.
Util
Contains a set of commonly used constants and macros in the header file.
