Overview

Base Test Framework
The Base Test Framework is a generic infrastructure for creating a test suite.
HPI Test Framework
The HPI Test Framework is an HPI specific infrastructure for creating an HPI test suite.
HPI Helpers
The HPI Helpers are a set of HPI specific classes that can be used by any of the test cases.
Utilities
The Utilities classes are set of generic non-HPI classes.
HPI Tests
Shows the tree-like hierarchy of the HPI test suite.
